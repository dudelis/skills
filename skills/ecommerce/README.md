# Ecommerce Skills

Skills for ecommerce product research and content preparation workflows.

| Skill                                                     | Description                                                                                                                                                                                                                                                                                          |
| --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [product-research](product-research/SKILL.md)             | Researches one skincare or dermocosmetics product, compiles a structured YuliSkin brief at `Products/[CompanyName]/[ProductName]/brief.txt`, and downloads 2 to 5 main product images into the same folder using sequential `[image-slug]-01` style names (lowercase ASCII handle slug).             |
| [product-create-shopify](product-create-shopify/SKILL.md) | Creates one new draft Shopify product from a product-research folder, using German default fields, best-effort English translations, existing metafields, handle generation, sequential product images, and duplicate-stop safety checks.                                                            |
| [company-research](company-research/SKILL.md)             | Researches one skincare or dermocosmetics brand, produces a Shopify-ready collection-page brief in German and English at `Collections/[CompanyName]/brief.txt`, downloads the brand logo, and discovers products from the brand site into `products.csv` for parallel fan-out to `product-research`. |
